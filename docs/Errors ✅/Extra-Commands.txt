Now run these commands in order:

python run.py

If application starts successfully, stop it with:

Ctrl + C

Then:

$env:FLASK_APP="run.py"

Verify Flask can load the app:

flask routes

If routes are displayed successfully:

flask db init

Then:

flask db migrate -m "Initial accounts table"

Then:

flask db upgrade

Finally start application:

python run.py

http://127.0.0.1:5000
