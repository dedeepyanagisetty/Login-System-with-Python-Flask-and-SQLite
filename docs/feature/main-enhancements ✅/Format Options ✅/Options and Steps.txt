Method 1 (Recommended)

Press:

Ctrl + Shift + P

Type:

Preferences: Open User Settings (JSON)

Open it.

If you see:

{
}

replace with:

{
    "editor.formatOnSave": true,

    "[python]": {
        "editor.defaultFormatter": "ms-python.black-formatter"
    }
}

Save:

Ctrl + S

Done ✅
